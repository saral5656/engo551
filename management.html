Colour Management

Consistency in colour both digitally and physically are incredibly important. A simple example of this is taking RGB images and converting to CMYK images for printing to reduce costly misprints. RGB and CMYK each can be described as a colour space and there a need to be able to convert from one to another. How is this done?

ICC (International Color Consortium) Profiles can be used in colour management to characterise data inputs and outputs on devices and allow for colour mapping, going from one colour space to another. Essentially CIELAB and CIEXYZ are used to convert the input space to the profile connection space and then to the desired output space. This is done with things like Bradford transformations, Hunt-Pointer-Estevez transformations, Stockman & Sharpe functions, and von Kries Transform Method.
